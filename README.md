# Invitación Digital - 15 Años Morena

## Descripción

web interactiva y totalmente responsive desarrollada para la invitación digital de 15 años. Presenta una estética oscura estilo neón/mística (tonos púrpura, rosa y magenta) con animaciones fluidas, tarjetas estilizadas en relación de aspecto móvil e integración directa con WhatsApp para la interacción con los invitados.

## Construido con

- React
- Vite
- Tailwind CSS

## Dependencias utilizadas

- react-hot-toast
- react-icons
- framer-motion

## Versionado

Git / GitHub

## Estructura de la invitación

### Menú / Header

Header flotante y navegación fluida entre secciones de la invitación.

### Secciones interactivas principales:

- **Hero:** Presentación principal.
- **Ubicación & Dress Code:** Información del salón, código de vestimenta y accesos directos a Google Maps / Waze.
- **Confirmación (RSVP):** Formulario para confirmar asistencia enviando los datos por WhatsApp.
- **Menú Especial:** Formulario interactivo para avisar sobre alergias o restricciones alimentarias.
- **Mensaje Especial:** Sección de dedicatorias para la cumpleañera.
